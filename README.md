# JUEGO PIEDRA PAPEL TIJERA
Para correr el juego en pyhton se debe ralizar el siguiente pasos
```sh
PASOS NECESARIOS
PRIMERA SE CREO LA CARPETA Y LUEGO EL ARCHIVO .PY
luego se enlazo con github
con los siguientes comandos
 - git init en el shell de git en la carpeta que se creo para desarrollar la aplicacion
 - luego se fue a la pagina google se busco gitingonre.io
 - se coloco los parametros para generar el archivo .gitignore
 - se cro el archivo readme.md
 - se creo en el git hub el repositorio y se copio el paramtro para enlazar 
 - git remote add origin https://github.com/JvicoC/PythonGitHubPlatzi.git y despues se vreifico con el comando en el shell git remote -v

 - en el shell se añadio los archivos con git add .
 - se hizo el respectivo commit git commit -m "añadiendo cambios"
 - se subio a github git push origin master
 -  y hasta ahi los cambios haciendo nuevamente git en cada cambio
```
